Biblioteca digital

Búsqueda organizada de materiales bibliográficos

Esta biblioteca integra un catálogo curado y una búsqueda externa de libros abiertos. Los recursos se presentan con datos bibliográficos básicos para facilitar la consulta, la selección y la citación inicial.

Los materiales del catálogo deben ser propios, autorizados, de acceso abierto o enlazados desde fuentes oficiales. Las credenciales o licencias institucionales no deben publicarse en una página abierta.

Organización sugerida

Modelo de crecimiento del módulo

La biblioteca debe crecer primero como catálogo curado y luego incorporar más fuentes externas. Para artículos académicos, se recomienda una fase posterior separada de Gutendex, ya que Gutendex está orientado a libros abiertos.

1. Catálogo curado

Registrar recursos reales con autor, año, tipo, fuente, formato, licencia, enlace, palabras clave y fecha de revisión.

2. Búsqueda externa

Usar Gutendex únicamente para libros abiertos, manteniendo los resultados como consulta externa.

3. Artículos académicos

Incorporar en una fase posterior una búsqueda separada para artículos, tesis o DOI, sin mezclarla con libros abiertos.

4. Mantenimiento

Revisar enlaces, actualizar fechas de revisión y retirar recursos vencidos o sin fuente clara.
